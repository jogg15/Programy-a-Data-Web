Softwarové licence

Pravidla určující, jak může být software používán a šířen.

Základní pojmy

Při pořízení programu si uživatel obvykle nekupuje samotný software, ale pouze licenci, tedy oprávnění k jeho používání. Program je chráněn autorským právem stejně jako jiné duševní dílo.

OEM licence je určena k prodeji spolu s hardwarem, například s počítačem nebo notebookem, a bývá cenově výhodnější než samostatná licence.

Multilicence umožňuje provozovat software na více zařízeních současně a využívá se hlavně ve firmách a školách.

Typy šíření softwaru

  • Demoverze: Program s omezenými funkcemi.
  • Trial verze: Plná verze dostupná pouze na omezenou dobu.
  • Shareware: Software určený k vyzkoušení, později zpoplatněný.
  • Freeware: Program zdarma, ale chráněný autorskými právy.


Svobodný a Open Source software

Open Source software má veřejně dostupný zdrojový kód, který může kdokoli studovat, upravovat nebo rozšiřovat. Nejčastěji je šířen pod licencemi typu GNU/GPL.

Oproti tomu proprietární software má uzavřený kód a jeho použití je striktně omezeno licenční smlouvou (EULA).

Příklady a modely

  • Public Domain: Software bez licenčních omezení.
  • GNU/Linux: Svobodný operační systém s otevřeným kódem.
  • SaaS: Software poskytovaný formou služby v cloudu.
