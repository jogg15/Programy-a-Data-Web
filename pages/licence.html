Softwarové licence

Právní rámec užívání softwaru a ochrana autorských děl

Základní pojmy

Uživatel si nekupuje program jako celek, ale získává licenci (oprávnění) k jeho užívání. Program je považován za autorské dílo a je chráněn autorským zákonem.

OEM licence: Software určený k zabudování do hardware, často prodávaný se zařízením za sníženou cenu.

Multilicence: Výhodné oprávnění provozovat program na více počítačích najednou (pro firmy a školy).

Typy šíření

  • Demoverze: Program se zablokovanými funkcemi (tisk, ukládání).
  • Trial: Plnohodnotný program, ale jen na omezenou dobu.
  • Shareware: Lze užívat po určitou dobu, poté je nutné zaplatit poplatek.
  • Freeware: Zdarma k užívání i rozšiřování při dodržení autorských práv.

Svobodný a Open Source software

Open Source znamená software s otevřeným zdrojovým kódem, který může kdokoli zkoumat, opravovat nebo vylepšovat. Musí být šířen pod licencemi jako GNU/GPL.

Komerční (proprietární) software: Má uzavřený kód a jeho užití omezuje licence EULA.

Příklady a modely

  • Public Domain: Programy k naprosto volnému užití.
  • GNU/Linux: Kombinace svobodného systému a jádra.
  • SaaS: Software jako služba pronajímaná v cloudu.
